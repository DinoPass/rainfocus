# RainFocus UI Challenge

Single-page UI challenge built with Next.js, React, and SCSS to match the provided RainFocus mock layout.

## Tech Stack

- Next.js
- React
- SCSS

## Run Locally

1. Install dependencies:

```bash
yarn
```

2. Start the development server:

```bash
yarn dev
```

3. Open `http://localhost:3000`

## Build / Export

To generate the static exported build:

```bash
yarn build
```

The exported files are included in the `build` folder.

## Preview Exported Build

To preview the exported build locally:

```bash
npx serve@latest build
```

## Notes

- The delivery package includes both source files and the exported `build` folder.
- Large generated folders such as `node_modules` and `.next` were removed for submission.
